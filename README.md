# computeEsppshares
This is tiny nodejs module, helps computing Espp shares based on your espp contribution, FMV value.

## Build
```
git clone https://github.com/MadhukarMoogala/computeEsppshares.git
cd computeEsppShares
npm run build
node indexB.js
```
